Angularjs Background

An angularjs directive
that generates a responsive background around your input
Choosen date is {{date1}} ...

Usage

Html

Wrap your input into the background element

Javascript

Add the background module dependency
angular.module('app', [ '720kb.background' ]);

Options

Date format

All the accepted date formats can be found here

Default date

You can set date to be displayed by default with date-set="" attribute data ( you can use all the accepted date formats by the javascript new Date())

Date limits

Setting date min and max limits its easy (you can use any javascript new Date() accepted date formats)

Customize buttons

Just a FontAwesome buttons example

Input as grandchild

Sometimes you cannot put date input as a first child of background. In this case you may use selector="" to point to the CSS class of the input. (below: Twitter bootstrap example)
