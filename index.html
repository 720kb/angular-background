Angularjs Background

Angular.js directive to dynamically switch
the background of your DOM elements
Paste an image url or for example a base64 encoded image

Usage

Html

Use the dynamic-background attribute to init the element

My dynamic background div

Javascript

Add the background module dependency
angular.module('app', [ '720kb.background' ]);

Options

Attributes

This is the same as writing Css background rules, so all the background rules are now attributes ( you can check all the supported attributes here)

My dynamic background div

Events

You can specify the background to change on a specific event or a set of events using the background-on-event="" data attribute

My dynamic background div that will change background on mouseover or keypress
